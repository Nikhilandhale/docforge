# Changelog

All notable changes to DocForge will be documented in this file.

## 0.1.0 - 2026-07-05

### Added
- Initial public MVP release of DocForge
- PDF upload endpoint with local file storage
- SQLite-backed document metadata persistence
- PDF parsing and text chunking with PyMuPDF
- Embedding generation and ChromaDB indexing
- Retrieval-based chat endpoint with source page references
- FastAPI OpenAPI documentation and Docker-based local setup
- Initial pytest coverage for the upload, parsing, and retrieval workflow
